# ☀️ SunTech Pakistan – Solar Energy Website

A modern, dark‑themed, glassmorphism landing page for a fictional solar company, built to showcase advanced frontend development skills.

**Live Demo:** [https://suntech-pakistan.vercel.app](https://suntech-pakistan.vercel.app)

## 🚀 Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** React Icons
- **Forms:** React Hook Form + Zod
- **Deployment:** Vercel (free hosting)

## ✨ Features

- **Fully Responsive** – Mobile-first design
- **Glassmorphism UI** – Frosted-glass cards, subtle borders, backdrop blur
- **Interactive Animations** – Parallax hero, staggered word reveals, animated counters, 3D tilt
- **Component Architecture** – Modular sections (Hero, Stats, Services, HowItWorks, Testimonials, CTA, TrustedBy)
- **Contact Form** – Validated with Zod, ready for email integration (Resend)
- **SEO Optimized** – Metadata API, semantic HTML
- **Performance** – Static generation (SSG) with Lighthouse 95+

## 📂 Pages

- **Home** – Full landing page with all sections
- **About** – Company story, values, timeline
- **Services** – Detailed service cards with feature lists
- **Contact** – Contact form with glassmorphism container and company details
